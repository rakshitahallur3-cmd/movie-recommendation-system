🎬 About This Project

This Movie Recommendation System is developed using Python, Flask, Machine Learning, TMDB API, HTML, CSS, and JavaScript.

Users can search for movies, receive personalized recommendations, watch trailers, save movies to their watchlist, and securely log in using Email or Google Authentication.

✨ Key Features

🎬 Movie Recommendations

Get similar movie suggestions using Machine Learning.

🔍 Smart Search

Search movies quickly with autocomplete suggestions.

▶ Watch Trailers

Watch official trailers directly from TMDB and YouTube.

❤️ Watchlist

Save your favourite movies for later viewing.

⭐ Movie Ratings

View ratings, release dates and movie details instantly.

🔐 Secure Login

Login using Email or Google Authentication securely.
