# Movie Recommendation System

A Flask-based Movie Recommendation System that recommends movies based on user selection using a machine learning model.

## Features
- User Login & Signup
- Movie Recommendations
- Movie Details
- Watchlist
- Responsive UI
- TMDB API Integration

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

## Installation

```bash
pip install -r requirements.txt
python app.py
